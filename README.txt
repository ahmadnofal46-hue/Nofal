# 📊 Dashboard Analisis Waktu FYP TikTok — @mlbb.cliperr

Versi web statis responsif (desktop & HP) — siap untuk diunggah ke **Vercel** atau **Netlify**.

---

## 🚀 Cara Deploy ke Vercel

1. Buka [https://vercel.com](https://vercel.com)
2. Login atau buat akun gratis.
3. Klik **Add New → Project**.
4. Pilih **Import Third-Party → Drag & Drop Folder**.
5. Seret folder ZIP ini setelah diekstrak (berisi index.html & app.js).
6. Klik **Deploy** — selesai! 🎉
7. Vercel akan memberi link seperti:
   ```
   https://fyp-dashboard.vercel.app/
   ```

---

## 🌍 Cara Deploy ke Netlify (alternatif mudah)

1. Buka [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag & drop folder hasil ekstrak ZIP ini.
3. Tunggu upload selesai → Netlify akan memberi link seperti:
   ```
   https://fyp-dashboard-netlify.app/
   ```
